# ESP8266 Temperature & Humidity Monitoring System

This project implements an IoT-based temperature and humidity monitoring system using ESP8266, DHT11 sensor, MQTT protocol, and Node-RED dashboard.

## Hardware Used
- ESP8266 (NodeMCU)
- DHT11 Temperature & Humidity Sensor

## Software & Tools
- Arduino IDE
- Node-RED
- MQTT (HiveMQ Public Broker)
- GitHub

## System Architecture
DHT11 → ESP8266 → MQTT Broker → Node-RED Dashboard

## Features
- Real-time temperature and humidity monitoring
- MQTT-based data communication
- Live gauges and graphs using Node-RED
- Scalable IoT architecture
